由于使用NDK开发，可以中间的数据放入到so中，这样是的关键数据更安全。因为破解原生代码相对来说太容易，而so文件相对来说门槛较高。
我们本篇就是从安全角度来使用NDK开发，将重要的数据放入NDK中，同时将重要的加密也放入到NDK开发，这样在一定程度上可以保证APP应用的安全。
