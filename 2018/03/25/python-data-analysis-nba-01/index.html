我们需要将1996-1997赛季到2016-2017赛季所有的按照月份给列出来，首先我们需要获取每隔赛季有哪几个月是比赛月(常规赛和季后赛)，因为每隔赛季的情况千差万别，如停摆，全明星赛，以及各种因素会导致每隔赛季有所差异。

那我们如何拿到每隔赛季的月份呢:

  1. 过滤2017-2018赛季（比赛尚未结束）
  2. 按照月份来去重，拿到有比赛的月份(比赛月)
  3. 每一个的比赛求总和
