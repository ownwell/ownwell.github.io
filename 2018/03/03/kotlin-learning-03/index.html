顶层的函数

顶层参数主要是为了解决类似utils工具类的问题，函数可以在类的顶层（不在类的内部），调用时，类似调用java的静态工具类。

对于java调用kotlin的顶层函数，需要注意导入的类是这个函数所在的类名加Kt后缀。
