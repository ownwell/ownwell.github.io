Sublime Text是一个跨平台的文本工具，对于大多数的开发者，大家并不陌生，我只是将我自己长常用的记录下来，好记性不如懒笔头。

SublimeText分为 Sublime Text2和Sublime Text3，其中3是收费版，会定时弹广告，不过我更喜欢3。其官网：http://www.sublimetext.com/

安装Package Control

Package Control是一个subltext的Package管理工具，通过它可以安装很多的插件。

可以通过其官网来搜索，相关插件。

安装步骤

打开控制台页面(windows是 ctrl +` mac是command+`)

在控制太输入安装Package Control的命令：

Sublime Text 3

Sublime Text 2

安装完，restart。

插件

通过弹出package control（command + shift+ P）–>输入install,选择Package Control 输入你要安装的插件。
